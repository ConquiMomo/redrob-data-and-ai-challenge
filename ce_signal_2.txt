Designed offline evaluation pipelines for ranking and retrieval systems using graded relevance judgements to compute rank-aware metrics on held-out query sets. Knew that optimising offline metrics does not guarantee better user engagement online, factoring in position bias and query distribution shift. Connected offline benchmarks to online A/B experiments and encountered cases where offline gains did not transfer. Built feedback loops from real user behavior back into the evaluation pipeline. Thinks about retrieval as a precision-recall problem with real users, not as a similarity score to maximise.
